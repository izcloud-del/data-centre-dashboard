# 🖥️ Data Centre Monitoring Dashboard

A simple web-based dashboard to simulate basic data centre monitoring.

## 📌 Overview
This project displays the status of multiple servers, including:
- Online / Offline status
- CPU usage
- Temperature levels
- Basic alert indicators (e.g. high load)

## 🎯 Purpose
To demonstrate foundational understanding of:
- Infrastructure monitoring
- System health visibility
- Operational awareness in data centre environments

## 🛠️ Tech Used
- HTML
- CSS

## 🚀 How to Run
1. Download or clone this repository
2. Open `index.html` in your browser
